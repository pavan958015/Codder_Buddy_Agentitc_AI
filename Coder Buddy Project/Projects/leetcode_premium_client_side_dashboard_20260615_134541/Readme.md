# LeetCode Premium Client-Side Dashboard

A responsive dashboard for tracking and analyzing LeetCode problem-solving stats and stats charts, built purely client-side with HTML, CSS, and JavaScript.

## 📂 Project Structure

- `index.html`: Main dashboard web layout.
- `style.css`: Visual styling sheets.
- `script.js`: Client-side logic and chart renders.

---

## 🚀 How to Run the Project

### 1. Open Terminal and Navigate to Project Directory
```powershell
cd "Coder Buddy Project/Projects/leetcode_premium_client_side_dashboard_20260615_134541"
```

### 2. Start a Local Static Web Server
Run a Python static server:
```powershell
python -m http.server 8000
```

### 3. Open in Browser
Open [http://localhost:8000](http://localhost:8000) in your web browser to view the dashboard interface.
