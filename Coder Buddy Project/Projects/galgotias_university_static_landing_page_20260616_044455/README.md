# Galgotias University Static Landing Page

A beautiful, responsive static landing page built for Galgotias University, detailing admissions, placement highlights, campus life, and inquiry workflows.

## 📂 Project Structure

- `index.html`: Main landing page interface.
- `src/styles/main_layout.css`: Stylesheet containing responsive layout configurations.
- `src/scripts/main_interactivity.js`: Client-side JavaScript.
- `config/database.py`: Boilerplate database setup file.
- `deployment/`: Containerization configuration files (`Dockerfile`, `docker-compose.yml`, `DEPLOY.md`).

---

## 🚀 How to Run the Project

### 1. Open Terminal and Navigate to Project Directory
```powershell
cd "Coder Buddy Project/Projects/galgotias_university_static_landing_page_20260616_044455"
```

### 2. Run the Static Page
Since this is a client-side frontend project:
- **Double click** the `index.html` file in your file explorer to open it in your browser.
- **Or run a local static server** inside the directory if you prefer:
  ```powershell
  python -m http.server 8000
  ```
  Then open [http://localhost:8000](http://localhost:8000) in your browser.
